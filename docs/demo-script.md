# VoiceCanvas AI 演示视频讲解脚本（2–3 分钟）

## 0:00–0:20 开场与定位

大家好，这是 VoiceCanvas AI，一款完全由中文语音驱动的智能矢量插画工具。它不依赖后端、收费 API 或生成模型，而是在浏览器本地把自然语言拆解为可解释、可撤销、可继续编辑的 Canvas 对象。

## 0:20–0:40 基础语音绘图

点击麦克风后说：

- “在 300 200 画一个半径 60 的绿色圆。”
- “画一条蓝色横线。”

讲解：基础图形能力仍然保留。右侧会实时显示结构化指令、对象图层、对象数量和执行耗时。

## 0:40–1:10 一句话生成夜晚城市

先说“清空画布”，再说“画一个夜晚城市”。

讲解：系统会生成渐变夜空、月亮、随机星点、不同高度的楼房、随机亮灯窗户和道路。日志会显示场景被拆解成了多少个稳定对象。所有随机参数已写入对象记录，重绘不会变化。

## 1:10–1:30 一句话生成海边日落

说“画一个海边日落”。

讲解：系统会生成暖色渐变天空、夕阳、海面、多层随机波浪、沙滩和椰子树剪影，展示场景模板能够生成完整插画，而不只是基础形状。

## 1:30–1:48 切换霓虹风格

说“切换为霓虹风格”，然后说“画一个机器人”。

讲解：霓虹风格会启用深色背景、亮色调色板和发光效果。页面顶部和右侧控制台会显示当前风格。

## 1:48–2:08 语音编辑最后一个图形

依次说：

- “复制最后一个图形。”
- “把最后一个图形向右移动。”
- “把最后一个图形变大。”
- “把最后一个图形变成蓝色。”

讲解：每次修改都会创建新的历史快照，因此仍然可以稳定撤销和重做。

## 2:08–2:28 撤销、重做与保存

说“撤销”，可以一次撤销上一次对象编辑；如果刚生成的是完整场景，撤销会一次移除整个场景。再说“重做”和“保存图片”，展示完整创作闭环。

## 2:28–2:55 架构与创新点

项目由语音控制器、中文规则解析器、指令执行器、场景工厂、风格系统、对象历史管理器和 Canvas 绘图引擎组成。场景级操作作为一个历史组，对象级编辑生成新快照；随机细节在创建时固化。

创新点是用零后端的方式实现“一句话生成完整矢量插画 + 继续语音编辑”，兼顾生成感、可解释性、稳定性和两天内可完成度。
