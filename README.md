# VoiceCanvas AI

VoiceCanvas AI 是一个语音控制的双模式智能绘图工具。用户可以选择稳定可编辑的 Canvas 矢量模式，也可以调用硅基流动 Kolors 模型生成表现力更强的完整 AI 图片。

## 核心亮点

- **Voice-first 交互**：绘图、撤销、重做、清屏、画笔设置和保存均可通过语音完成。
- **双绘图模式**：Canvas 模式负责即时矢量绘图，AI 图片模式负责生成完整图片。
- **安全 AI 代理**：API Key 仅保存在本地或 Vercel 环境变量中，不进入浏览器构建产物。
- **中文规则解析器**：支持颜色、形状、操作同义词，以及中文数字和不完整指令的默认参数。
- **智能矢量插画场景**：一句“画一个夜晚城市 / 星空 / 海边日落 / 森林 / 机器人”可生成完整插画。
- **组合式自然语言绘图**：从一句话中同时识别场景、时间、天气、风格和多个素材关键词，自动组合画面。
- **可扩展素材库与自动布局**：三十余种矢量素材按背景、远景、主体和点缀自动分层布局。
- **五种全局风格**：默认、霓虹、极简、手绘、柔和风格会影响背景、调色板、线宽、透明度和发光。
- **稳定对象模型**：每个对象保存 ID、标签、位置、尺寸、颜色、风格和创建时间，可继续语音修改。
- **原子历史记录**：复杂图形作为一个操作组进入历史，一次撤销即可完整移除。
- **实时反馈**：展示结构化指令、执行结果、响应耗时、对象数量、最近对象和场景拆解数量。
- **现代前端仪表盘**：使用 React、TypeScript、Vite、Tailwind CSS、Canvas API 与 Web Speech API；仅 AI 图片调用经过轻量安全代理。

## 如何运行

ES Module 通常需要通过本地 HTTP 服务打开，不建议直接双击 `index.html`。

```bash
npm install
npm run dev
```

然后访问 `http://localhost:8000`，允许浏览器使用麦克风。生产构建可运行 `npm run build`。

仅使用 Canvas 模式不需要 API Key。使用 AI 图片模式时，复制 `.env.example` 为 `.env` 并填写：

```env
SILICONFLOW_API_KEY=你的密钥
SILICONFLOW_IMAGE_MODEL=Kwai-Kolors/Kolors
SILICONFLOW_IMAGE_SIZE=1024x1024
SILICONFLOW_INFERENCE_STEPS=20
SILICONFLOW_GUIDANCE_SCALE=7.5
```

部署到 Vercel 时，在项目环境变量中配置相同变量。不要提交真实 `.env` 或 API Key。

## 双模式使用

### Canvas 矢量模式

默认进入 Canvas 模式，继续支持基础形状、组合场景、对象编辑、撤销、重做和保存。切换模式不会丢失 Canvas 作品与历史。

### AI 图片模式

点击顶部“AI 图片”，或说 `切换到 AI 绘图模式`。AI 模式采用“先描述、后确认”的交互：

1. `画一座漂浮在云端的未来城市`
2. `再加一些飞船，切换为赛博朋克风格`
3. 点击画布右上角或右侧控制台中的 `确认并生成`

AI 模式支持手动点击 `确认并生成`、`重新生成`，并支持语音 `撤销`、`重做`、`清空画布`、`保存图片`、`取消本次修改` 和 `切换到 Canvas 模式`。语音只负责维护提示词，图片 API 请求必须由用户点击按钮确认，避免语音识别分段或误识别造成重复生成。修改会合并到提示词，再手动确认生成整张图片，不是真实局部重绘。

## 推荐浏览器

推荐使用最新版 **Google Chrome** 或 **Microsoft Edge**。Web Speech API 的实现和可用性由浏览器提供；Firefox、Safari 或部分网络环境可能无法使用语音识别。

## 支持的语音指令

### 基础绘图

- `画一条红色线`
- `画一条蓝色横线`
- `画一条黑色竖线`
- `从 100 100 到 300 300 画一条红色线`
- `画一个红色圆`
- `画一个半径 50 的蓝色圆`
- `在 300 200 画一个半径 60 的绿色圆`
- `画一个宽 200 高 100 的绿色矩形`
- `在 200 150 画一个红色长方形`
- `画一个黄色三角形`

### 编辑与设置

- `清空画布`、`清屏`、`擦掉`
- `撤销`、`后退一步`、`上一步`
- `重做`、`恢复`、`下一步`
- `设置颜色为红色`
- `设置线宽为 5`
- `保存图片`、`导出图片`、`下载图片`

### 复杂创作

- `画一个房子`
- `画一个笑脸`
- `画一个太阳`

### 高级场景指令

- `画一个夜晚城市`、`画一座城市`、`画一个城市夜景`
- `画一片星空`、`画一个宇宙星空`
- `画一个海边日落`、`画一个日落海滩`
- `画一片森林`、`画一个森林`
- `画一个可爱的机器人`

高级场景会拆解成几十到上百个稳定对象，包括渐变背景、星点、建筑、窗户、道路、波浪、山丘、树木和机器人组件。整个场景作为一个历史步骤，撤销时一次移除。

### 组合式绘图指令

- `画一个有太阳、白云、草地、房子和小河的春天场景`
- `画一个夜晚校园，有月亮、星星、教学楼和树`
- `画一个海边，有太阳、船和海鸥`
- `画一个公园，有树、花、草地、长椅和小路`
- `画一个太空场景，有星星、月亮、飞船和行星`
- `画一个雨天城市，有道路、车、楼房和雨滴`
- `画一个儿童画风格的生日场景，有气球、笑脸、爱心和花`

解析器会生成包含 `scene`、`time`、`weather`、`style` 和 `objects` 的结构化 `composeScene` 指令。用户明确说出的素材优先绘制，场景预设会补充必要环境对象。

### 追加绘制与全局调整

- `加一个太阳`、`加几朵云`、`再加一些树`
- `再加一条小河`、`再加几颗星星`、`加一点花`
- `让画面更丰富一点`
- `让画面更亮一点`、`让画面更暗一点`
- `让画面更温暖`、`让画面更冷一点`

追加素材不会清空原画面，每次追加和全局调整都会作为独立历史步骤加入撤销/重做。

### 风格切换

- `切换为默认风格`
- `切换为霓虹风格`
- `切换为极简风格`
- `切换为手绘风格`
- `切换为柔和风格`
- `切换为儿童画风格`
- `切换为水彩风格`
- `切换为像素风格`

### 语音编辑最后一个对象

- `删除最后一个图形`
- `复制最后一个图形`
- `把最后一个图形变大`、`把最后一个图形变小`
- `把最后一个图形向左移动`、`向右移动`、`向上移动`、`向下移动`
- `把最后一个图形变成红色`、`把最后一个图形变成蓝色`

颜色支持红、蓝、绿、黄、黑、白、紫、橙及其“红色 / 红的”等常见说法。形状支持圆、圆形、圆圈、矩形、长方形、方形、线、直线、横线、竖线、三角形、三角。

## 默认参数

不完整指令会自动补全：当前颜色、当前线宽、画布中心附近的位置、圆半径 50、矩形 160 × 100、线长 180、三角形大小 120。

## 常见问题

**点击开始后没有反应？**  
确认使用 Chrome 或 Edge，并通过 `http://localhost:8000` 打开项目。点击开始后，允许地址栏弹出的麦克风权限；如果之前选择过拒绝，请点击地址栏左侧的站点权限图标，将麦克风改为允许，然后刷新页面。语音识别服务还可能受网络环境影响。

**为什么坐标没有生效？**  
坐标建议清晰停顿，例如：“在 三百、二百，画一个圆”。识别结果中两个坐标需要能被区分。

**为什么响应耗时与体感不同？**  
页面展示的是收到最终识别文本后，解析和执行指令的耗时。浏览器返回语音识别结果之前的时间不受项目控制。

**清空画布后能撤销吗？**  
可以。清空也进入历史快照，可通过“撤销”恢复。

## 演示建议

1. 展示基础圆形和带坐标图形，证明基础能力仍然稳定。
2. 说“画一个夜晚校园，有月亮、星星、教学楼和树”，展示组合式关键词解析。
3. 说“再加一些花和气球”，展示在当前画面上追加素材。
4. 说“切换为儿童画风格”，展示全局风格变化。
5. 再展示亮暗冷暖调整、最后对象编辑、撤销、重做和保存。

完整讲解脚本见 [`docs/demo-script.md`](./docs/demo-script.md)。

## 两天开发范围说明

本项目优先保证本地可运行、演示稳定、架构清晰和核心交互完整。Canvas 模式使用可解释的中文规则解析器驱动智能矢量插画；AI 图片模式通过轻量安全代理调用硅基流动官方生图接口与 Kolors 模型，但不在本地运行 Diffusion、ControlNet 或 LLM Agent。
